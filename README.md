rlox
====
`rlox` is a Rust implementation of `jlox`, the tree-walk interpreter from the [Crafting Interpreters](http://www.craftinginterpreters.com/) book.

Running
-------

Clone the project to a local directory. Build and run the project using cargo.

```text
https://github.com/AndyPhuHuynh/lox-rust
cd lox-rust
cargo run
```
